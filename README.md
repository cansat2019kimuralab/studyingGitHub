# GitHub 練習用リポジトリ

## 概要

GitHubに慣れるための学習用リポジトリです。
基本的に自由に好き勝手に使っていいです。


## GitHubを始めるうえでダウンロードしておいたほうがいいもの

- GitHub Desktop
	- ファイルの操作をGUI上でできる
- Atom
	- エディタ
	- C系の作業もAtomで完結する
	- IDEの外部エディタとして使っていいかも
